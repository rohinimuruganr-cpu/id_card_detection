# id-_card-_detection
Detect and recognize ID Card using YOLOv8, Webcam and uploaded images

## Project Description
This project detects and recognizes ID Cards using YOLOv8.

## Features
- Detect ID cards
- Upload images
- Webcam detection
- Fast recognition

## Technologies Used
- Python
- YOLOv8
- OpenCV

## How to Run
1. Install Python
2. Install libraries
3. Run:
python test.py.py
